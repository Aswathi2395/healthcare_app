# Healthcare Web App

## 📌 Overview
This is a simple healthcare support web application that allows users to request help and get basic assistance through a chatbot.

## 💻 Tech Stack
- HTML
- CSS (Bootstrap)
- JavaScript

## 🤖 AI / Automation Idea
The app includes a basic chatbot that responds to common healthcare queries such as:
- emergency help
- doctor consultation
- general support

It uses simple keyword-based logic to simulate an AI-powered FAQ assistant.

## 🏥 NGO Use Case
This application can be useful for NGOs and healthcare organizations to:
- Collect patient support requests through a form
- Provide quick automated responses using a chatbot
- Assist users during emergencies
- Improve accessibility to basic healthcare support
